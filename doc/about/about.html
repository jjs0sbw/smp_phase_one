General Structural Modeling Application

General ---- Version 0.01.00
This web application is designed to support the testing and improvement of
structural modeling techniques and methods. Structural modeling is used to create
a system structure from a disorganized and unordered set of objects. To properly
test the structuring methods and techniques, a system with known order is selected
and scrambled into a ramdom system state. This ramdom, unstructured system is then
structured using the methods and techniques associated with structural modeling. When
the ordering process is complete, the final output from the process is checked against
The known system structure.

This web application supports structural modeling by organizing and presenting a web
based interface to a set of general structural modeling grids. Theses general structural
modeling grids support a range of system structuring approaches. At this time, the
application supports three distinct structuring approaches. These three appraoches are
based on the software commands from the GMU ISM software.
